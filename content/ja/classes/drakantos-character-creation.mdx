ロスターは3つの主要な役割にわたってバランスが取れています。最初のクローズドベータテストでは限られた選択肢でしたが、フルリリースではこのリストを大幅に拡大することを目指しています。各ヒーローには固定された基本属性が付属していますが、その究極の可能性は、ゲームの進行システムとどのように相互作用するかによって形作られます。

<Section icon={Palette} title="視覚的なカスタマイズと染色">

ヒーローの顔の構造や身長を変更することはできませんが、**Drakantosのキャラクター作成**は、コスメティックシステムを通じて視覚的な選択肢を提供します。開発者は、プレイヤーがゲームの2Dピクセルアートの世界で目立つことができるように、パーソナライゼーションのレイヤーを実装しています。

カスタマイズの主要な方法は**染色システム**です。これにより、ヒーローのスプライト上の特定の16進数カラーゾーンを変更できます。

| カスタマイズの種類 | 説明 | 入手方法 |
| :--- | :--- | :--- |
| **鎧の染色** | 装備の主要色と副次色を変更します | クラフト、ドロップ、または冒険者ギルド |
| **髪の染色** | ヒーローの基本の髪色を変更します | バニティショップまたは特別イベント報酬 |
| **コスチューム** | 装備の外見を上書きする全身スキン | ゲーム内ストアまたはレアなダンジョンドロップ |
| **バニティアイテム** | 帽子や背中の装飾品などの小さなアクセサリー | アチーブメント報酬とギルドトークン |

    - **コレクション**: グローバルなバニティライブラリで追跡されます。
  </Card>
</CardGrid>

<Callout type="warning" title="染色の消費">
染料は通常、使い捨てアイテムです。珍しい色を見つけた場合は、使用すると消費されるため、適用する前にヒーローの選択に満足していることを確認してください。
</Callout>

<Section icon={Sparkles} title="オーブとアーティファクトによるスキルカスタマイズ">

Drakantosでの真の「創造」は、選択画面の後に行われます。このゲームは、ヒーローのアビリティの機能方法を変更できるモジュラースキルシステムを利用しています。これは、**オーブ**と**アーティファクト**という2つの主要なアイテムによって処理されます。

レベル12で、ヒーローは独自のクラススキル（タンクの挑発やサポートの主要な回復など）をアンロックします。この時点から、アクティブスキルに直接装着できるオーブを見つけ始めます。

| アイテムタイプ | 装着方法 | ゲームプレイへの影響 |
| :--- | :--- | :--- |
| **スキルオーブ** | 特定のアビリティにソケット | ダメージ増加、クールダウン短縮、または効果追加 |
| **アーティファクト** | 2つの専用装備スロット | 新しいアクティブ/パッシブスキルを付与（例：ハチの巣DOT） |
| **強化** | 鍛冶屋で適用 | 装備の基本パワーと装備スコアを向上 |

<Callout type="success" title="レベル12のパワースパイク">
レベル12に到達することは、キャラクターの初期段階で最も重要なマイルストーンです。この時点から、戦術的なダンジョン攻略に必要なスキルが完全に揃い、初期戦闘の「硬さ」が解消されます。
</Callout>

アーティファクトは、ダンジョンを出るたびにリフレッシュされる「サブスキル」として機能します。たとえば、タンクは敵を攻撃させるアーティファクトを装備して、基本キットにはない追加の群衆制御を提供できます。これにより、「ブロウラーサポート」や「タフなDPS」のようなハイブリッドビルドが可能になります。

</Section>

<Section icon={Shield} title="装備スコアと装備の進行">

Drakantosでは、キャラクターの強さは**装備スコア**で表されます。この数値は、高ティアダンジョンでの効果を決定する上で、実際のキャラクターレベルよりも重要です。装備システムは、4つの主要スロットと2つのアーティファクトスロットに簡素化されています。

| スロット | 影響 | 主な入手源 |
| :--- | :--- | :--- |
| **武器** | ダメージ出力とスキル倍率 | ボスドロップとクラフト |
| **帽子** | 体力とマナのプール | ダンジョンチェスト |
| **上半身** | 物理防御と魔法防御 | クエスト報酬とドロップ |
| **靴** | 移動速度と回避クールダウン | オープンワールドでの採集/クラフト |

<Steps>
  <Step num="1" title="ティア1ダンジョンを周回">
    3人制のインスタンスをプレイして、基本的な装備のフルセットを集めましょう。すべての敵を倒して戦利品のドロップを最大化することに集中してください。
  </Step>
  <Step num="2" title="リソースを収集">
    オープンワールドに向かい、鉱石を採掘し、木を伐採しましょう。これらの材料は、鍛冶屋で装備を強化するために必要です。
  </Step>
  <Step num="3" title="装備スコアを向上">
    集めた材料を使ってティア1の装備をアップグレードしましょう。これにより、より良い経験値率のティア2ダンジョンへのアクセスがアンロックされます。
  </Step>
  <Step num="4" title="オーブをソケット">
    スキルオーブを見つけたら、すぐにソケットに装着しましょう。装備とは異なり、オーブはヒーローのアビリティに永続的なブーストを提供します。
  </Step>
</Steps>

<Callout type="info" title="戦術的な移動">
