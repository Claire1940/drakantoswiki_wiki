<Section icon={Swords} title="戦闘メカニクスとスキルカスタマイズ">

**ドラカントス イリュージョニスト**の戦闘は、タイミングとポジショニングの微妙なバランスです。ボタン連打を好む現代のアクションRPGとは異なり、ドラカントスはリアルタイム環境内で意図的で、ほとんどターン制のようなペースを取ります。この「昔ながら」の感覚は、すべての呪文詠唱と移動コマンドが大きな重みを持つことを意味します。現在のところ複雑なマナ管理が不足しているため、プレイヤーはクールダウンを効果的に管理する必要があり、ほとんどの遭遇戦はクールダウンのローテーションのテストとなります。

ヒーローのカスタマイズは、オーブとアーティファクトという2つの主要なシステムを通じて行われます。これらのアイテムを使用すると、ソロプレイまたはグループダイナミクスのどちらかに合わせてアビリティを調整できます。

| アイテムタイプ | 機能 | 使用制限 |
| :--- | :--- | :--- |
| **オーブ** | 既存のスキルプロパティを変更する | 交換するまで永続的 |
| **アーティファクト** | アクティブなユーティリティ/アイテムを提供する | ミッションごとに使用回数制限あり |
| **装備** | 基本パワースコアを増加させる | アップグレードには精錬が必要 |

    - 地中に潜る敵や脅威度の高いターゲットに高ダメージの呪文を集中させる。

<Section icon={Target} title="進行とパワーレベリング">

**ドラカントス イリュージョニスト**として進むには、冒険者ギルドのランクを上げる必要があります。あなたのランクは、アイテムレベルやギアスコアと同様に機能する「パワー」スコアに直接結びついています。進行はレベルだけではなく、装備の品質と強化が重要です。

<Steps>
  <Step num="1" title="ギルドクエストを完了する">
    ランク1と2のミッションから始めて、基本的な緑と青の装備を集めます。
  </Step>
  <Step num="2" title="精錬と強化">
    鍛冶屋を訪れて不要なアイテムを精錬し、そのリソースを使用して主要な装備を強化します。
  </Step>
  <Step num="3" title="アーティファクトの獲得">
    ユーティリティスロットを埋めるためにアーティファクト（毒樽や体力回復など）を集めます。
  </Step>
  <Step num="4" title="ランクアップ">
    パワーが必要なしきい値に達したら、ランクアップ試練を完了して、より難しいダンジョンをアンロックします。
  </Step>
</Steps>

| 装備のレア度 | ドロップ元 | 精錬価値 |
| :--- | :--- | :--- |
| **コモン (白)** | 標準的な雑魚敵 | 低 |
| **アンコモン (緑)** | ランク1以上のミッション | 中 |
| **レア (青)** | ランク2以上のミッション | 高 |
| **エピック (紫)** | スペシャルイベント/ボス | 非常に高い |

<Callout type="success" title="効率アップのヒント">
ソロでミッションをこなす場合、モンスターのHPがフルパーティよりも低く設定されることが多いため、パワースコアが要件をわずかに上回っている場合は、より速く装備を稼ぐことができます。
</Callout>

</Section>

<Section icon={Trophy} title="PvPとエンドゲーム競技">

PvEで**ドラカントス イリュージョニスト**をマスターしたプレイヤーのために、PvPシステムはバランスの取れた戦場を提供します。League of LegendsやBrawl Starsのような競技タイトルに触発されたドラカントスのPvPでは、ステータスが正規化されるため、スキルと戦略のみが重要になります。これにより、より高いパワーを持つプレイヤーが、より熟練した相手に対して不公平な優位性を持つことがなくなります。

- **ランクシステム：** ラダーを登り、限定タイトルとスキンを獲得します。
- **週刊トーナメント：** ギルドベースまたはソロトーナメントに参加して、希少な収集品を獲得します。
- **ゲームモード：** チームデスマッチや目的ベースのマップなど、様々なモードがあります。

| PvPモード | チームサイズ | 焦点 |
| :--- | :--- | :--- |
| **アリーナ** | 1v1 / 3v3 | 純粋な戦闘スキル |
| **目的** | 3v3 | マップ制御と戦略 |
| **トーナメント** | 可変 | トーナメント形式の勝ち抜き戦 |

<Callout type="info" title="スキル表現">
イリュージョニストの敵を移動させ、デコイを作成する能力は、エリア否定が鍵となる3v3の目的モードで非常に価値のある資産となります。
</Callout>
