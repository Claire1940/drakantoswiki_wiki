<Section icon={Swords} title="コアアビリティの内訳">

トーマスのキットは、「強き調べ」と「音の結界」というコンセプトを中心に設計されています。各アビリティのニュアンスを理解することは、パーティーへの貢献を最大化するために不可欠です。**ドラカントスの吟遊詩人**は、敵の陣形を崩しながら味方を生き残らせるために、正確なタイミングとポジショニングに依存します。

| アビリティ名 | 主な効果 | 主要なユーティリティ |
| :--- | :--- | :--- |
| **通常攻撃** | 単体魔法ダメージ | オーブによる高速詠唱速度 |
| **スライド** | 5%DR付き前方ダッシュ | シールドと速度バフ |
| **強き調べ** | 3つの魔法の音符を発射 | 精密ヒットで追加ダメージ |
| **音の結界** | DR/AOEダメージで味方をバフ | 再発動でノックバックを含む |
| **レゾナンス** | 4コードのリズムシーケンス | AOEスリープとチームシールド |
| **癒しの歌** | 8秒間のAOE回復オーラ | 近接ベースのモバイル維持 |

<Callout type="tip" title="戦闘戦略">
常に**音の結界**を前線のタンクに使用してください。ダメージ軽減を提供するだけでなく、パルス状のAOEダメージはアグロを維持し、小規模な雑魚敵を排除するのに役立ちます。
</Callout>

**スライド**アビリティは、トーマスの主要な生存ツールです。標準的なダッシュとは異なり、5%のダメージ軽減（DR）が内蔵されており、オーブによって強化することで最大HPの15%の巨大なシールドを提供できます。これにより、トーマスはキャスタークラスのヒーローとしては驚くほど耐久性があります。

</Section>

<Section icon={Target} title="レゾナンスメカニクスの習得">

**ドラカントスの吟遊詩人**の決定的な特徴は、**レゾナンス**アビリティです。このスキルは、他の人気MMOの「ダンサー」ステップに似たミニゲームを導入します。レゾナンスを成功させるには、画面に表示される4つのコードのシーケンスを入力する必要があります。

<Steps>
  <Step num="1" title="発動">
    レゾナンススキルを発動し、コードシーケンスUIを呼び出します。ゲームは4つの特定の色の音符を表示します。
  </Step>
  <Step num="2" title="コード入力">
    表示された順序で対応するボタンを正確に押します。ここでは速度が重要であり、クールダウンはパフォーマンスに応じて変化します。
  </Step>
  <Step num="3" title="スリープ発動">
    完璧に完了すると、トーマスはAOEダメージを与え、近くのすべての敵を眠らせ、重要な休息時間を提供します。
  </Step>
  <Step num="4" title="シールド蓄積">
    最後のオーブが装備されている場合、シーケンスの成功した各ステップは、近くのすべての味方にシールドの層を追加します。
  </Step>
</Steps>

<Callout type="warning" title="入力エラー">
コードを間違えると通常はエラーになり、効果がキャンセルされます。ただし、最初のオーブアップグレードでは1回のミスが許容されるため、リズムを学ぶ初心者には非常にお勧めです。
</Callout>

レゾナンスは特に**護衛ミッション**で効果的です。新しい敵の波が出現したとき、タイミングの良いレゾナンスはパック全体を眠らせることができ、護衛対象が安全に通過する間、パーティーは優先ターゲットに集中できます。

</Section>

<Section icon={Sparkles} title="オーブのカスタマイズと進行">

ドラカントスにおけるオーブは、ヒーローのアビリティを専門化することを可能にします。トーマスにとって、オーブは彼を単純なヒーラーからユーティリティの強力な存在へと変えます。適切なオーブの選択は、セーフティネットとクールダウン短縮を追加することで、彼のリズムメカニクスの「ギミック感」を大幅に軽減できます。

| スキル | オーブ1の効果 | オーブ2の効果 |
| :--- | :--- | :--- |
| **スライド** | 追加チャージ/調整されたCD | 距離とDR持続時間の増加 |
| **通常攻撃** | 詠唱速度+15% | 詠唱速度+25% |
| **強き調べ** | 中央ヒットボーナスノート | 手動照準速度の向上 |
| **音の結界** | 敵の移動速度低下 | 再発動時のノックバック |
| **レゾナンス** | 1回のミス許容 | 速度に基づくCD短縮 |

<Callout type="success" title="ハイレベルビルド">
エンドゲームコンテンツでは、**レゾナンスオーブ2**を優先してください。クールダウン短縮により、AOEスリープを連鎖させることができ、これは現在のドラカントスメタで最も強力なCCと言えるでしょう。
</Callout>

レゾナンスの最後のオーブは、味方に最大HPの15%のシールドも追加します。癒しの歌と組み合わせることで、トーマスは入ってくるバーストダメージを効果的に「無効化」し、同時にパーティーのHPバーを最大にすることができます。

</Section>
