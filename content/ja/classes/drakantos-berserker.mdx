<Section icon={Shield} title="アクションコンバットと移動メカニクス">

ハイブリッドシステムから100%アクションコンバットシステムへの移行は、PvEとPvPの両方で**Drakantos バーサーカー**の機能に根本的な変化をもたらしました。すべての基本攻撃と特殊能力は手動で照準を合わせる必要があります。左クリックで標準コンボを実行し、右クリックボタンを長押しすると強化された特殊攻撃が発動します。バーサーカーの場合、これはしばしば大規模な範囲攻撃や、小規模なNPCのグループを一掃できる強力な前方突進となります。

| メカニクス | キーボード＆マウス | コントローラーサポート |
| :--- | :--- | :--- |
| **移動** | 8方向移動 | 完全な360度流体移動 |
| **ターゲティング** | マウスカーソルで照準 | 右アナログスティックで照準 |
| **攻撃** | 左クリック（基本） / 右クリック（特殊） | トリガーベースの入力 |
| **精度** | スキルショットに高い精度 | 高い機動性と快適性 |
| **複雑さ** | PCベテランにとって習得が容易 | 360度照準の習得には時間が必要 |

<Callout type="warning" title="重要">
現在のベータビルドでは、ターゲティングシステムが「粘着質」に感じられる場合があります。キャラクターが正しい方向を向いていない場合、スキルがマウスの方向には発動しないことがあります。長いチャージアニメーションを開始する前に、常にキャラクターのモデルが脅威の方向を向いていることを確認してください。
</Callout>

移動はもはやグリッドに制限されません。あらゆる方向にダッシュや回避が可能で、これは予兆のあるボス攻撃を避ける必要がある近接クラスにとって不可欠です。Drakantosの近接攻撃には「スタミナのような」特性があります。クールダウンなしで使用できますが、2秒ごとに攻撃のタイミングを計ることが、キャラクターの戦闘フローを枯渇させることなく高いDPSを維持する最も最適な方法です。

</Section>

<Section icon={Map} title="PvPモードと競技戦略">

Drakantosでは、プレイヤーが望めばPvPのみで完全に進行することができます。これはMMORPGジャンルにとって大きな変化であり、従来のクエストに代わる実行可能な選択肢を提供します。**Drakantos バーサーカー**は、拠点防衛や敵の構造物に対する積極的な攻撃が必要なモードで特に強力です。

<Callout type="info" title="ゲームバランス">
PvPでは、すべてのプレイヤーが最大レベルの装備でスケーリングされます。これにより、スキルとビルドの相乗効果が、時間のかかるグラインドよりも重要となる公平な競技場が確保されます。
</Callout>

アサルトモードは最も複雑で、タワーの共有ヘルスプールと、3分30秒ごとに湧く中央バフが特徴です。このバフは60秒間攻撃力を25%増加させ、敵のコアをバーストダウンしようとするバーサーカーにとってはゲームチェンジャーとなります。コントロールと護衛モードは試合時間が短いため、実績と報酬はより早く獲得できますが、アサルトモードは1試合あたり最も多額の報酬を提供します。

</Section>

<Section icon={Sparkles} title="ビルドの最適化: アーティファクトとオーブ">

**Drakantos バーサーカー**の進行は、レベルアップだけではありません。ゲームには、アビリティを修正するオーブ、アーティファクト、トロフィーの奥深いシステムがあります。ベータ期間中、プレイヤーは通常なら数週間かかるようなハイエンドビルドをテストするために、特別なベータボックスを受け取ります。

| アイテムの種類 | 機能 | 保証メカニクス |
| :--- | :--- | :--- |
| **オーブボックス** | ヒーロー固有のアビリティ修飾子を付与 | 3つの選択肢のうち1つは現在のヒーロー用 |
| **アーティファクトボックス** | 受動的な戦闘ボーナスを提供 | 3つのアーティファクトのランダム選択 |
| **トロフィーボックス** | エンドゲームのステータスブーストを提供 | 3つのトロフィーのランダム選択 |
| **スキルオーブ** | スキルの挙動を変更（例: パイロファイアオーブ） | キャラクターメニューから適用する必要がある |

<Callout type="success" title="ビルドのヒント">
オーブボックスを開く際は、常にオーブが欲しいヒーローとしてプレイしていることを確認してください。ゲームは、3つの選択肢のうち少なくとも1つがアクティブなキャラクターと互換性があることを保証します。
</Callout>

バーサーカーのビルドでは、ヘルス再生やダメージ軽減を提供するアーティファクトを探してください。例えば、「ウォータースキン」アーティファクトは、戦闘後10秒間毎秒40のヘルスを提供し、プル間のダウンタイムを大幅に短縮します。また、パイロギャザーファイアオーブのようなスキル修飾子も見つけることができ、これは狭い範囲に80ダメージを与え、クモの巣の駆除やダンジョン内のNPCグループを一掃するのに最適です。

</Section>
