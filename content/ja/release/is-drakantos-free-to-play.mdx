<Section icon={Coins} title="収益化とプレイヤーエコノミー">

ゲームは無料プレイですが、Winen Game Studiosはライブサービスのアップデートを維持する必要があります。収益化戦略は、水平的な進行と美学に焦点を当てることで、「ペイ・トゥ・ウィン」の落とし穴を避けるように設計されています。プレイヤー主導のマーケットプレイスでは、ほとんどのアイテムを取引できますが、リアルマネートレード（RMT）を防ぎ、経済的安定を維持するために特定の制限が存在します。

| 機能 | 無料プレイアクセス | オプションのサブスクリプション/有料 |
|:---|:---|:---|
| **コアコンテンツ** | すべてのストーリークエストとダンジョン | 制限されたストーリーコンテンツなし |
| **ヒーロー名簿** | さまざまな無料ヒーローへのアクセス | コスメティックスキンとエモート |
| **マーケットプレイス** | 完全な取引機能 | 税金割引または追加スロットの可能性 |
| **進行** | 標準的なXPとルート率 | 利便性向上ブースト（戦闘外） |
| **PvPアリーナ** | ランクモードへのフルアクセス | 限定シーズンコスメティック |

開発者は長期的な開発資金を賄うためのサブスクリプションモデルを検討していますが、競技プレイには必須ではないことを強調しています。ヒーローの力を決定するアーティファクトとレリックは、直接購入するのではなく、ゲームプレイを通じて獲得されます。

<Callout type="warning" title="マーケットプレイスのセキュリティ">
インフレやボット行為を防ぐため、開発者は特定の高価値アイテムの取引に厳格な制限を設け、真のプレイヤーにとって健全な経済を維持しています。
</Callout>

Drakantosの最も特徴的な機能の1つは、伝統的なクラスシステムの廃止です。プレイヤーは「戦士」や「魔術師」を選ぶ代わりに、個々のヒーローの名簿から選択します。各ヒーローはユニークなボイスアクト、アニメーション、そしてエルドラースの世界における特定の役割を持っています。このアプローチは「ヒーローシューター」やMOBAに似ていますが、永続的なMMORPGの世界に適用されています。

ヒーローは、3つの異なるシステムを使用してさらにカスタマイズできます。
1. **オーブ**: 特定のアビリティの特性を変更します。
2. **アーティファクト**: 強力なアクティブまたはパッシブ効果を付与します。
3. **レリック**: ヒーローのキットの機能方法を根本的に変更します。

<Callout type="info" title="名簿管理">

<Section icon={Swords} title="アクションコンバットと移動のオーバーホール">

初期のプレイテストでは、元のタイルベースの移動が現代のプレイヤーにとって硬く感じられることが明らかになりました。これに対応して、開発者は大規模なオーバーホールを行い、ゲームを**自由移動システム**に移行させました。この変更により、コントローラーでの360度移動と、より流動的なマウスエイムコンバットが可能になりました。戦闘は完全にアクションベースになり、プレイヤーはスキルショットを狙い、迫りくる攻撃を手動で回避する必要があります。

| 戦闘要素 | 旧システム（2026年以前） | 新しいオーバーホールシステム |
|:---|:---|:---|
| **移動** | タイルベース / 8方向 | 完全自由 / 360度 |
| **ターゲティング** | ハイブリッド / タブターゲット要素 | 100%アクション / エイムスキル |
| **基本攻撃** | 2秒固定クールダウン | スタミナのような「チャージ」システム |
| **ダッシュ** | 固定距離タイル | 方向性のある流動的なダッシュ |
| **インタラクション** | アニメーションロック | より良いカイトのためのロックインの減少 |

<Callout type="success" title="コントローラーサポート">
新しい移動システムは、コントローラーに特に最適化されており、右アナログスティックでキャラクターの向きを操作し、左スティックで移動を操作することで、遠距離ヒーローにとって「ツインスティック」のような感覚を生み出します。
</Callout>

</Section>
