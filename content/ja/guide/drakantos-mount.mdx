### 準備と前提条件

厩舎は主要な町の東に少し歩いたところにあります。もし到着してもサリーが話してくれない場合は、以下の要件と現在のステータスを確認してください。

| 要件 | 値 | 注釈 |
| :--- | :--- | :--- |
| **メインストーリーパワー** | 460+ | メインキャンペーンミッションを進める |
| **冒険ランク** | 様々 | サイド目標やダンジョンを完了してレベルアップする |
| **NPCの場所** | サリーの厩舎 | スタート地点の町の宿屋の東 |
| **クエスト名** | 迷子のロバを見つける | 資格を得ると青いマーカーとして表示される |

<Callout type="tip" title="プロのヒント">
パワー460に到達してもすぐにクエストマーカーが表示されない場合は、宿屋に戻って冒険ランクの報酬を確認してみてください。ランクアップ通知がサリーとの会話の最終的なトリガーとなることがあります。

<Section icon={Target} title="迷子のロバを探す">

サリーがあなたを信用するようになると、彼女は自慢のロバが迷子になったことを教えてくれます。ここから「迷子のロバを見つける」クエストが始まります。マップ上には大きな青い検索サークルが表示されますが、そのエリアはかなり広大で、ロバは多くのプレイヤーが見落とす特定の場所に隠れています。

### ロバの場所を見つける

ロバはクエストサークルの最南端に位置しています。厩舎から南に主要道路を進み、大きな交差点に到達します。そこからさらに南に進んで高レベルの地域に入るのではなく、少し西へ向かいます。すると、森の端近くにロバが立っているのを見つけるでしょう。

### 誘導メカニクス

ロバを厩舎まで誘導するのは、しばしばクエストの最もイライラする部分として挙げられます。ロバは従来のペットのように機能せず、独自の経路探索ロジックを持ち、不適切に扱われると「逃げ出す」傾向があります。

| アクション | 結果 | 最適な使い方 |
| :--- | :--- | :--- |
| **近くにいる** | ロバがあなたについてくる | 主要道路のまっすぐな道に使う |
| **後ろから押す** | ロバを誘導する | 道を外れた場合に修正するのに使う |
| **遠すぎる** | クエストがリセットされる | ロバを画面の境界線から決して離さない |
| **障害物** | ロバが立ち往生する | 低木やフェンスを通り抜けさせるのではなく、迂回させる |

<Steps>
  <Step num="1" title="交差点を見つける">
    青いクエストゾーンの南の交差点に向かいます。
  </Step>
  <Step num="2" title="最初のインタラクション">

  <Step num="3" title="ゆっくりと護衛する">

  <Step num="4" title="厩舎に入る">
    サリーの建物の後ろにある囲いまでロバを誘導します。成功すると「チン」という音が聞こえます。
  </Step>
</Steps>

<Callout type="info" title="ナビゲーションの注意">
ロバのAIは道路をたどることを好みます。森の中を近道しようとすると、ロバは迷子になったり、地形に引っかかったりして、クエストが再開される可能性が非常に高くなります。時間がかかっても主要な道を進みましょう。

<Section icon={Swords} title="ロバの調教ミニゲーム">

ロバを無事に連れ帰った後、サリーが乗ることを許可する前に調教する必要があります。これにより調教ミニゲームが導入されます。このメカニクスは将来の**ドラカントス マウント**捕獲にも再利用される可能性があります。このフェーズでは、範囲攻撃（AoE）の配置とリズムに基づいたインタラクションの組み合わせが必要です。

### 調教の戦略と操作

目標は、ロープでロバを捕まえ、そして引き寄せることです。しかし、ロバは臆病で、気を散らすものを使わない限り積極的にあなたを避けます。

<CardGrid cols={2}>
  <Card title="小石投げ">
    小石（デフォルトのインタラクション）を使ってロバを驚かせます。これにより、ロバは囲いの向こう側を予測可能な方向に走るようになり、ロープで捕まえやすくなります。
  </Card>
  <Card title="ロープキャッチ">
    ロープボタン（デフォルトは 'I' キー）を押してAoEサークルを表示します。このサークルをロバの進路の*前*に配置し、ロバが直接罠に走り込むようにする必要があります。
  </Card>
</CardGrid>

### 引き寄せのメカニクス

ロバがロープに捕らえられると、「引き寄せ」フェーズが始まります。ここでほとんどのプレイヤーは攻撃的になりすぎて失敗します。ロバを自分の方に引き寄せるには、リールボタン（デフォルトは 'O' キーまたはコントローラーのLB）を押し続ける必要があります。

| ロバの状態 | プレイヤーの行動 | 結果 |
| :--- | :--- | :--- |
| **落ち着いている / 動いていない** | リールを保持 (O/LB) | 進行度が増加し、ロバが近づく |
| **抵抗している / 走っている** | リールを離す | ロープが切れるのを防ぐ |
| **ロープが赤くなっている** | 直ちに停止 | 高い張力はロープを切り、ゲームをリセットする |

<Callout type="warning" title="重要な警告">
ロバが積極的に引っ張っている間は、リールボタンを押し続けないでください。ロープの色に注意し、明るい赤色になったら、調教の試みが失敗するまでにボタンを離す時間は1秒未満です。
</Callout>

</Section>

<Section icon={Trophy} title="サリーとのライセンスレース">

調教されたロバがいても、ライセンスなしではエルトラスで合法的に乗ることはできません。サリーは、交通量の多い場所でマウントを扱う協調性があることを証明するために、あなたにレースを挑みます。このレースは驚くほど難しく、限られたブーストメカニクスを賢く使う必要があります。

### レースコースの基本

レースは一連の旗に従って進みます。旗を1つでも見逃すと、サリーがミスをすることはめったにないため、通常は負けになります。コースには直線路、低木の周りの急カーブ、そして狭い橋があります。

**レース戦略チェックリスト:**
- **ブーストを温存する**: スプリント/ブーストは長い直線路でのみ使用します。
- **コーナリング**: カーブ中はスプリントを避けてください。高速になるとロバの旋回半径が大きくなり、障害物にぶつかりやすくなります。
- **サリーのパターン**: サリーは通常、スタート時と最後の最後でスプリントします。最終直線で互角の場合は、ブーストバーの少なくとも50%を残しておいてください。
- **ブロック**: サリーの真ん前に留まることで物理的にブロックできますが、勢いを失うと危険です。

| レース区間 | 推奨戦略 | ブースト使用量 |
| :--- | :--- | :--- |
| **スタート** | サリーの尾に密着する | 最小限 |
| **森の道** | 旗の精度に集中する | なし |
| **橋** | 狭い道; 追い越さない | なし |
| **最終直線** | ゴールまで全力疾走 | 最大限 |

<Callout type="success" title="勝利のヒント">
繰り返し負けている場合は、低木の配置を記憶してみてください。低木にぶつかると著しく減速し、このような短いレースでは回復がほぼ不可能になります。
</Callout>

</Section>

<Section icon={Sparkles} title="マウントの装備と使用">

レースに勝利すると、サリーはあなたにマウントライセンスを付与します。しかし、ロバは自動的にキャラクターの下に現れるわけではありません。ライセンスを手動で有効にし、UIを通じてマウントを装備する必要があります。

### アクティベーションの手順

**Drakantosマウント**の設定を完了するには、以下の特定の手順に従ってください。

<Steps>
  <Step num="1" title="インベントリを開く">
    'B'キーを押してメインインベントリ画面を開きます。
  </Step>
  <Step num="2" title="ライセンスを見つける">
    「三つ星」タブ（スペシャルアイテム）に移動します。マウントライセンスを見つけて選択し、Enterキーを押して消費します。
  </Step>
  <Step num="3" title="コスメティクスメニュー">
    'Q'または'E'を押してキャラクタータブを循環させ、「コスメティクス」セクションに到達します。
  </Step>
  <Step num="4" title="マウントを割り当てる">
    右側にある小さな黄色の点が付いた空のタイルを探します。このタイルをクリックし、ロバを選択して確認します。
  </Step>
</Steps>

### 操作と使用方法

一度装備すると、屋外エリアならいつでもマウントを召喚できます。

- **乗降**: 'F'（デフォルト）を長押し。
- **移動**: 標準のWASDキー。
- **スプリント**: スペースバー（スタミナ/ブーストバーを使用）。
