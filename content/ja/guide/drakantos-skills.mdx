<Section icon={Shield} title="マーリーの防御および基本スキル">
マーリーの主要な防御アビリティであるディバインプロテクションは、彼女のキットの要です。基本攻撃のホーリーライトは、安定したダメージを提供します。これらのスキルは、オーブで強化されることで、基本機能以上の役割を果たすことができます。

### ディバインプロテクション (Divine Protection)
マーリーの防御アビリティであるディバインプロテクションは、自身にシールドを張り、ダメージを吸収します。オーブの改造により、その実用性を劇的に高めることができます。

| オーブレベル | 効果 | 説明 |
| :-------- | :----- | :---------- |
| 基本 | 最大体力の20%相当のシールド | 壊れるか再発動するまでマーリーを保護します。 |
| オーブ 1 | ダメージ軽減 +30% | シールドがアクティブな間、受けるダメージを軽減します。 |
| オーブ 2 | 味方をターゲット | 選択した味方にシールドを付与できるようになります。 |
| オーブ 3 | シールドブレイク・ブラスト | シールドが壊れた際、聖なる光の爆発（106ダメージ）を放ち、周囲の敵をノックバックさせます。 |
| オーブ 4 | 4秒間の無敵状態 | 詠唱開始時に4秒間の無敵状態を付与します。 |

### ホーリーライト (Holy Light - 基本攻撃)
マーリーの基本攻撃であるホーリーライトは、ダメージを与えるシンプルな光の矢です。そのオーブ改造は、詠唱速度の向上に焦点を当てています。

| オーブレベル | 効果 | 説明 |
| :-------- | :----- | :---------- |
| 基本 | 42 ダメージ | ターゲットに光の矢を放ちます。 |
| オーブ 1 | 詠唱速度 +10% | ホーリーライトを詠唱する速度を上げます。 |
| オーブ 2 | 詠唱速度 +25% | 攻撃をさらに速くするため、詠唱速度をさらに向上させます。 |

<Callout type="warning" title="ベータ版のため変更の可能性があります">
**Drakantosのスキル**とそのオーブ改造、特にサポートおよび防御ロールについては、開発の進行に伴い変更される可能性があります。開発者は、非DPSロールがより早くそのユーティリティにアクセスできるように、これらを再配置することを積極的に検討していると述べています。

<Section icon={Swords} title="攻撃および範囲攻撃（AoE）スキル">
マーリーはまた、チャスタイズやホーリーノヴァのように、攻撃に貢献し敵を制御することを可能にするアビリティも持っています。これらのスキルは、敵の集団を管理し、戦闘中の味方をサポートするために不可欠です。

### チャスタイズ (Chastise)
チャスタイズは、敵を拘束（ルート）する前方範囲攻撃（AoE）であり、重要なクラウドコントロールを提供します。

| オーブレベル | 効果 | 説明 |
| :-------- | :----- | :---------- |
| 基本 | 89 ダメージ、1秒間の拘束 | 前方範囲にダメージを与え、敵の動きを封じます。 |
| オーブ 1 | チャージでダメージ増加 | アビリティを長押ししてチャージすることで、ダメージを増加させます。 |
| オーブ 2 | チャージで拘束時間延長 | チャージすることで、拘束効果の持続時間も延長されます。 |
| オーブ 3 | チャージ時間短縮 | チャスタイズをフルチャージするのに必要な時間を短縮します。 |

### ホーリーノヴァ (Holy Nova)
ホーリーノヴァは、マーリーの周囲の敵にダメージを与え、移動速度を低下させる強力な範囲アビリティです。そのオーブ強化により、チームにとって重要なユーティリティへと変化します。

| オーブレベル | 効果 | 説明 |
| :-------- | :----- | :---------- |
| 基本 | 範囲ダメージ、60%移動速度低下(4秒) | 周囲のすべての敵にダメージを与え、移動速度を低下させます。 |
| オーブ 1 | チャージで味方にシールド | チャージすると周囲の味方にディバインプロテクションを付与し、15秒間最大体力の20%のシールドを与えます。 |
| オーブ 2 | 範囲拡大、ノックバック | 爆発半径を広げ、ノックバック効果を追加します。クラウドコントロールに最適です。 |

<Callout type="success" title="クラウドコントロールのシナジー">
チャスタイズの拘束とホーリーノヴァの鈍化、そして潜在的なノックバックを組み合わせることで、マーリーは敵の大群を効果的にコントロールし、チームメイトがダメージを与えたり位置を立て直したりする機会を作り出すことができます。このシナジーは、彼女の**Drakantosのスキル**ツールキットの重要な側面です。
</Callout>

<Section icon={Target} title="究極のサポートおよび回復アビリティ">
マーリーのアルティメットアビリティであるガーディアンライトとセリエルの光は、献身的なサポートキャラクターとしての彼女の役割を確固たるものにします。これらのスキルは、重要な保護と持続的な回復を提供し、味方が最も過酷な遭遇を生き延びることを確実にします。

### ガーディアンライト (Guardian Light)
ガーディアンライトは、一時的な無敵状態を付与した後に爆発し、周囲の敵にダメージを与える単体対象の防御アビリティです。

| オーブレベル | 効果 | 説明 |
| :-------- | :----- | :---------- |
| 基本 | 100%ダメージ軽減(2秒)、爆発 | 選択した味方に2秒間の100%ダメージ軽減を付与し、その後に爆発します。 |
| オーブ 1 | 鈍化解除、移動速度上昇 | 鈍化効果を除去し、効果中の2秒間、味方の移動速度を上昇させます。 |
| オーブ 2 | 命中した敵ごとにシールド | 爆発が命中した敵1体につき、マーリーの最大体力の5%に相当するシールド（4秒間持続）を付与します。 |

### セリエルの光 (Seriel's Light)
セリエルの光はマーリーの究極の回復アビリティであり、一定時間、範囲内に継続的な回復を提供します。

| オーブレベル | 効果 | 説明 |
| :-------- | :----- | :---------- |
| 基本 | 継続回復(10秒) | 継続的な回復を提供するエリアを設置します。開始時が最も強く、徐々に減衰します。 |
| オーブ 1 | 回復範囲拡大 | セリエルの光の効果範囲を広げ、より多くの味方を回復できるようにします。 |
| オーブ 2 | リジェネ（継続回復）バフ | エリアを離れた味方も、5秒間少量の継続回復バフを受け続けます。 |

<Callout type="info" title="戦略的な配置">
セリエルの光の有効性は、**戦略的な配置**に大きく依存します。特に長期戦において、味方が持続的な回復の恩恵を受けられる場所に配置することで、戦況を一変させることができます。最大の効果を得るために、チョークポイントやチームが集まる場所を検討してください。
</Callout>
