<Section icon={Target} title="マップマーカーと自動経路探索の使用方法">

MMORPGでは効率が鍵であり、**Drakantosインタラクティブマップ**は、ピクセルアートジャンルで最も堅牢な自動経路探索システムの一つを提供します。このシステムにより、キャラクターが移動している間、インベントリの管理やギルドメンバーとのチャットに集中できます。

<Steps>
  <Step num="1" title="目的地を特定する">
    ワールドマップ（M）を開き、目的地のアイコンを見つけます。アイコンは色分けされています：クエストは黄色、サービスは青、戦闘ゾーンは赤です。
  </Step>
  <Step num="2" title="自動経路探索を開始する">
    マップ上の場所を左クリックするだけです。ミニマップ上に経路が描画され、ヒーローは自動的に目標に向かって移動を開始します。
  </Step>
  <Step num="3" title="障害物を監視する">
    自動経路探索は信頼できますが、混雑した場所でのNPCや他のプレイヤーからの「ボディブロック」には注意してください。開発者は現在、これらの衝突を最小限に抑えるために経路探索を改善しています。
  </Step>
  <Step num="4" title="到着とインタラクション">
    到着すると、自動経路探索は停止します。その後、NPCと交流したり、マップインターフェースから直接インスタンスアドベンチャーに入ることができます。
  </Step>
</Steps>

自動経路探索システムは、広大なカステロン市をナビゲートする際に特に役立ちます。都市のレイアウトは複雑であるため、マップを使用して鍛冶屋や錬金術師のような特定のベンダーを見つけることで、大幅な時間を節約できます。

<Callout type="success" title="探索報酬">
マップ上の新しいエリアを発見すると、「冒険者XP」が付与されることが多く、これは全体のアカウントレベルに貢献し、新しいティアのミッションをアンロックします。
</Callout>

| マーカータイプ | アイコンの外観 | 用途 |
| :--- | :--- | :--- |
| **メインクエスト** | 大きな金の感嘆符 | 主要なストーリー進行ポイント。 |
| **サイドクエスト** | 小さな銀の感嘆符 | 追加のリソースとXPのためのオプションタスク。 |
| **アドベンチャー** | ポータル/ゲートアイコン | 3人用インスタンスミッションへの入り口。 |
| **マーケット** | 天秤アイコン | プレイヤーとアイテムを取引できる場所。 |
| **ボス出現** | スカルアイコン | エリートワールドボスが出現する可能性のあるエリアを示します。 |

</Section>

<Section icon={Swords} title="PvPマップとアリーナナビゲーション">

Drakantosは探索だけではありません。**Drakantosインタラクティブマップ**には、3つの異なるモードにわたる13の専用PvPマップも含まれています。オープンワールドとは異なり、これらのマップは対称的で、競技のバランスのために設計されています。

<CardGrid cols={2}>
  <Card title="アリーナモード">
    - **コントロール**: 特定のゾーンを占領し、保持します。
    - **エスコート**: クリスタルを敵拠点まで導きます。
    - **アサルト**: タワーを使ったMOBAスタイルのレーンプッシュ。
  </Card>
  <Card title="マップレイアウト">
    - 隠れるための垂直性や草むらを特徴とする。
    - 3:30間隔で動的なバフが出現します。
    - 公平性を確保するための対称的なデザイン。
  </Card>
</CardGrid>

PvPでは、マップは重要な戦術情報を提供します。**エスコート**モードでは、ミニマップがクリスタルの正確な経路を追跡し、**アサルト**では、タワーとコアの残りHPが表示されます。

| PvPモード | マップの焦点 | 勝利条件 |
| :--- | :--- | :--- |
| **コントロール** | 中央ゾーン | すべてのポイントを完全に占領して勝利。 |
| **エスコート** | 直線経路 | クリスタルを最終目的地まで移動させる。 |
| **アサルト** | レーンとジャングル | 敵のコアを破壊し、タワーを守る。 |

<Callout type="info" title="PvPスケーリング">
PvPマップに入ると、すべてのヒーローは最適化された装備で最高レベルにスケーリングされます。これにより、勝利は装備のグラインドではなく、スキルとマップ認識によって決定されます。
</Callout>

</Section>
