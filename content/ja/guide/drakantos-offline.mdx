<Section icon={Shield} title="Drakantosがオンライン接続を必要とする理由">
Drakantosのオンライン専用の性質は、そのデザインと提供する機能に不可欠です。ゲームを定義する多くの要素は、オフライン環境では機能しません。これらの側面を理解することで、**Drakantosオフライン**オプションがなぜ実現不可能であるかが明確になります。

| 機能                | オンライン依存関係                                | 説明                                                                     |
| :------------------ | :------------------------------------------------ | :----------------------------------------------------------------------- |
| **大規模多人数参加型**  | MMORPGジャンルの核                                | 共有された世界で何千人ものプレイヤーと交流します。                       |
| **オンライン協力プレイ＆PvP** | 他のプレイヤーとのリアルタイム通信                  | 友達とダンジョンに参加したり、様々なPvPモードで競い合ったりします。      |
| **マーケットシステム**    | プレイヤー主導の経済とアイテム取引                  | 他のオンラインプレイヤーとレアアイテムや収集品を取引します。             |
| **ギルドシステム**      | 協力的なプレイヤー組織                            | ギルドに参加または作成し、ギルド固有の活動やランキングに参加します。     |
| **グローバルイベント**    | サーバーサイドの時限アクティビティとチャレンジ      | 特定のサーバー時間で発生するミニゲームやイベントに参加します。           |
| **ダイナミックダンジョン** | 常に変化する敵、ボス、目的                        | ミッションは時間とともに進化し、サーバー全体の進行状況に影響される可能性があります。 |
| **昼夜サイクル**      | サーバー時間に基づいて敵やチャレンジに影響を与える  | ゲーム内の世界は独自の時間サイクルに反応し、ゲームプレイに影響を与えます。 |
| **スキルカスタマイズ**  | オーブやアーティファクトがアビリティを変更          | カスタマイズはローカルですが、取得とメタはオンライン主導です。         |

<Section icon={Target} title="最小システム要件と接続性">
プレイヤーが準備できるように、Drakantosの公式Steamページにはシステム要件が明確に記載されており、特にインターネット接続の必要性が言及されています。

<Steps>
  <Step num="1" title="オペレーティングシステムを確認する">
    システムが**Windows 7以降**または**Linux 64ビット**で動作していることを確認してください。これらがDrakantosでサポートされているオペレーティングシステムです。
  </Step>
  <Step num="2" title="ハードウェア仕様を確認する">
    最小ハードウェア要件を満たしていることを確認してください：**Intel Core i3プロセッサ**、**4 GB RAM**、および**1GBビデオメモリ（シェーダーモデル3.0以降）**。
  </Step>
  <Step num="3" title="インターネット接続を確保する">
    重要なのは、**ブロードバンドインターネット接続**があることを確認することです。これは、WindowsとLinuxの両方で「プレイにはインターネット接続が必要」と明示的に記載されています。
  </Step>
  <Step num="4" title="ストレージスペースを割り当てる">
    ゲームのインストール用に少なくとも**1 GBの空きストレージスペース**があることを確認してください。
  </Step>
</Steps>

| 要件               | Windows 最小要件                     | Linux 最小要件                       |
| :----------------- | :----------------------------------- | :----------------------------------- |
| **OS**             | Windows 7+                           | Linux 64 bit                         |
| **プロセッサ**       | Intel Core i3                        | Intel Core i3                        |
| **メモリ**           | 4 GB RAM                             | 4 GB RAM                             |
| **グラフィックス**     | 1GB ビデオメモリ、シェーダーモデル 3.0+  | 1GB ビデオメモリ、シェーダーモデル 3.0+  |
| **DirectX**        | バージョン 9.0c                      | N/A                                  |
| **ネットワーク**       | プレイにはインターネット接続が必要     | ブロードバンドインターネット接続     |
| **ストレージ**       | 1 GB の空き容量                      | 1 GB の空き容量                      |

<Callout type="info">
ゲームはグラフィック的にそれほど負荷が高くないかもしれませんが、サーバーとの絶え間ないデータ交換には、ラグや切断を防ぐための信頼性の高いインターネット接続が必要です。これは、特にPvPやダンジョンでのゲームプレイ体験に深刻な影響を与える可能性があります。
</Callout>
