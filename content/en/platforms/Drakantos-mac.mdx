<Section icon={Map} title="How to Play Drakantos on Mac: Step-by-Step">

Since there is no native DMG installer, you will need to create a "Bottle" or a virtual environment to run the Windows executable. The following steps focus on using Whiskey, which is currently the most optimized free solution for **Drakantos Mac** users in 2026.

<Steps>
  <Step num="1" title="Download and Install Whiskey">
    Download the latest version of Whiskey from its official repository. Drag the application into your Applications folder and launch it to initialize the necessary Wine binaries.
  </Step>
  <Step num="2" title="Create a New Bottle">
    Click the "+" icon to create a new bottle. Name it "Drakantos" and ensure the "Windows Version" is set to Windows 10 or 11. This ensures compatibility with the game's modern Steam dependencies.
  </Step>
  <Step num="3" title="Install Steam for Windows">
    Download the Windows Steam installer (`SteamSetup.exe`). In Whiskey, click "Run" and select the installer. Follow the prompts to install Steam within your new bottle.
  </Step>
  <Step num="4" title="Configure Game Porting Toolkit">
    Within the bottle settings, toggle "DXVK" or "D3DMetal" to ON. Since Drakantos uses DirectX 9.0c, DXVK often provides the most stable frame rates for 2D pixel-art rendering.
  </Step>
  <Step num="5" title="Launch Drakantos">
    Open Steam inside the bottle, log in, and search for Drakantos. Install the game as you would on a PC. Use the "Play" button within Steam to start your adventure.
  </Step>
</Steps>

<Callout type="tip" title="Performance Optimization">
If you experience stuttering during high-action PvP matches, disable "Retina Mode" in your bottle settings. This lowers the internal rendering resolution, significantly boosting FPS without noticeably affecting pixel-art quality.
</Callout>

</Section>
