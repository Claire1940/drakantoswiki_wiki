<Section icon={Swords} title="Action Combat and Movement Overhaul">

Early playtests revealed that the original tile-based movement felt stiff to modern players. In response, the developers completed a massive overhaul, transitioning the game to a **free-movement system**. This change allows for 360-degree movement on controllers and more fluid mouse-aimed combat. The combat is now fully action-based, requiring players to aim skill shots and dodge incoming attacks manually.

| Combat Element | Old System (Pre-2026) | New Overhauled System |
|:---|:---|:---|
| **Movement** | Tile-based / 8 directions | Fully free / 360-degree |
| **Targeting** | Hybrid / Tab-target elements | 100% Action / Aimed skills |
| **Basic Attacks** | 2-second fixed cooldown | Stamina-like "Charge" system |
| **Dashing** | Fixed distance tiles | Directional fluid dashes |
| **Interaction** | Animation-locked | Reduced lock-in for better kiting |

<Callout type="success" title="Controller Support">
The new movement system is particularly optimized for controllers, where the right analog stick manages character facing while the left stick handles movement, creating a "twin-stick" feel for ranged heroes.
</Callout>
